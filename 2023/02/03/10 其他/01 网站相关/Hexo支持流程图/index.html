Hexo支持流程图

龙云 2023-02-03 13:04:06
Categories： 其他 > 网站相关 Tags： Markdown

  • 有时候需要用流程图来说明思路，使用图片的话，加载太慢，使用markdown的语法来画流程图是一个不错的办法。

  • vscode自带的markdown预览功能似乎不支持显示流程图，需要安装一个插件Markdown Preview Enhanced

  • hexo中默认不支持流程图的markdown语法，需要安装相关的插件来进行渲染。

流程图模块

  • mermaid

    • https://github.com/webappdevelp/hexo-filter-mermaid-diagrams

  • flowchart

    • https://github.com/bubkoo/hexo-filter-flowchart

  • sequence

    • https://github.com/bubkoo/hexo-filter-sequence

不同的模块，安装和使用方法也不相同，这里我安装mermaid

安装Mermaid

步骤1
 npm i hexo-filter-mermaid-diagrams --save
步骤2

修改Hexo配置文件_config.yml

# mermaid chart
mermaid: ## mermaid url https://github.com/knsv/mermaid
  enable: true  # default true
  version: "9.3.0" # default v7.1.2
  options:  # find more api options from https://github.com/knsv/mermaid/blob/master/src/mermaidAPI.js
  #startOnload: true  // default true
步骤3

在主题目录里，有个footer.ejs文件（一般在layout目录下面），把下面代码加到最后

<% if (config.mermaid && config.mermaid.enable && page.mermaid) { %>
  <script src="https://cdn.bootcdn.net/ajax/libs/mermaid/<%= config.mermaid.version %>/mermaid.min.js"></script>
    </script>
<% } %>

步骤4

把mermaid: true 加入页面文件

---
title: Hexo支持流程图
date: 2023-02-03 13:04:06
tags: Markdown
mermaid: true
---

Mermaid功能

Mermaid支持以下图形：

  • Flowchart 流程图

  • Sequence Diagram 时序图

  • Class Diagram 类图

  • State Diagram 状态图

  • Entity Relationship Diagram 实体关系图

  • User Journey 用户旅程图

  • Gantt 甘特图

  • Pie Chart 饼图

  • Requirement Diagram 需求图

  • Gitgraph Diagrams Git分支图

另外还有几种实验性质的图形，不一定能用

  • C4 Diagrams C4图

  • Mindmap 迷你地图

Mermaid使用方法

把流程图代码用 ```mermaid ``` 包裹起来。

1. 流程图 mermaid flowchart TB/TD/BT/RL/LR

流程图方向

  • TB - top to bottom 从上到下

  • TD - top-down/ same as top to bottom 从上到下，和TB一样

  • BT - bottom to top 从下到上

  • RL - right to left 从右到左

  • LR - left to right 从左到右

流程图基本单元

有方形框 、圆角框、菱形等十几种节点，节点间的连接线也有好几种，详见 流程图基本单元。
